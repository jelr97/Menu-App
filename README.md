# Menu App

Mobile-first meal planning app with recipes, weekly planner, and grocery lists.

## Features

- Browse 17 starter recipes (bilingual EN/ES)
- Add your own custom recipes
- Adjust servings with automatic ingredient scaling
- Edit individual ingredient amounts
- Plan meals for the week with flexible time slots
- Auto-generate grocery lists from your meal plan
- Add manual items to your grocery list
- Export/import your data as JSON
- Works offline after first load
- No dependencies — pure HTML, CSS, JavaScript

## Usage

Open `index.html` in any browser, or visit the GitHub Pages URL.
